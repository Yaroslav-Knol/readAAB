# Читать книги Алисы Бейли и Тибетца СЕЙЧАС! / Read AAB NOW!

**Приложение для чтения книг Алисы А. Бейли и Тибетца (Джуал Кхула) со случайной страницы для Android (APK) и Web. А теперь и поиска по книгам Алисы Бейли! 😀**

An app for reading books by Alice A. Bailey and The Tibetan (Djwhal Khul) starting from a random page. Available for Android (APK) and Web.

**Веб-версия:** [Ryurik.space/records/search/read](https://ryurik.space/records/search/read)  
**Скачать APK (Android):**[readAABNOW_v1.0.apk](https://ryurik.space/readAABNOW_v1.0.apk)

## О приложении

**Как часто ты хочешь обратиться к Учению, но страдаешь от «свободы выбора» 🤣 — какую книгу открыть, какую главу выбрать?**

Главная функция приложения — мгновенно отвести в *неслучайное* место из 26 книг Алисы Бейли и Тибетца. Это идеальный инструмент для ежедневных размышлений, медитаций и получения духовных знаний. 

Помимо случайного чтения, в приложение **интегрирован мощный поисковик**, который позволяет искать нужные термины и цитаты по всей базе книг.

## Возможности | Features

* Одно нажатие — и ты в нужном месте.
* Навигация по главам относительно текущей.
* Выбор темы и размера шрифта.
* Современная редакция (опционально): классические эзотерические термины заменяются на современные и понятные: *«Лучи» → «Потоки»*, *«Ученики» → «Ученики (серверы)»*. Новые термины осадил **Учитель ВМ Рюрик Олег Юрьевич**.
* Офлайн-база. Все тексты надежно упакованы под капотом.

## Встроенная поисковая система | Advanced Search Engine

Приложение оснащено мощным алгоритмом полнотекстового поиска, специально адаптированным для работы со сложными текстами. Поисковик доступен отдельно в [веб-версии на Ryurik.space](https://ryurik.space/records/search/trita).

### Как искать (Синтаксис запросов)
* **Простой поиск:** Просто введи нужное слово или фразу (например, `Иерархия` или `Космический Огонь`).
* **Поиск нескольких слов в одном контексте:** Используй разделитель `|` (вертикальная черта), чтобы найти абзацы, в которых одновременно встречаются несколько разных понятий или предложений. 
  * *Пример запроса:* `любовь|воля|активный интеллект` — алгоритм найдет только те фрагменты текста, где присутствуют все три *строки* одновременно.

### Параметры и фильтры поиска
1. **Свободные окончания (Морфологический поиск):** 
   * *Включено по умолчанию.* Умный алгоритм автоматически отсекает русские окончания слов (падежи, склонения, спряжения). Если ты ищешь слово `Учитель`, система найдет также `Учителя`, `Учителю`, `Учителем` и т.д. Это позволяет не упустить важные цитаты из-за разницы падежей.
2. **Досимвольный поиск (Точное совпадение):**
   * Ищет ровно то сочетание букв, которое ты ввёл, игнорируя другие словоформы. Удобно для поиска специфических аббревиатур или точных цитат.
3. **Область поиска (Фильтрация по книгам):**
   * Ты можешь искать как по всей базе сразу, так и выбрать конкретные книги (например, искать упоминания Сириуса только в «Эзотерической астрологии»).

### Отображение результатов и Экспорт
* **Автоматический глоссарий (компиляция):** Результаты выдаются не просто ссылками, а целыми осмысленными абзацами (сниппетами), чтобы ты мог сразу понять контекст цитаты.
* **Цветовая подсветка:** Искомые слова подсвечиваются в тексте для быстрого чтения по диагонали (при поиске через `|` разные слова могут подсвечиваться разными цветами).
* **Сохранение в TXT:** Ты можешь скачать результаты своей исследовательской выборки в виде текстового файла (`.txt`). Файл будет содержать все найденные абзацы с указанием книги-источника и оригинальной ссылки на сайт. Удобно для создания собственных конспектов и компиляций!

## English Summary

**Read AAB NOW!** is a non-commercial reading and search tool designed for students of Esotericism, Theosophy, and the teachings of Alice A. Bailey and Djwhal Khul (The Tibetan). 

Bypass the paradox of choice by opening a random section of the teachings for your daily meditation. Includes adjustable reading themes, font sizes, a full-text search engine with morphological support, and an optional "Modern Edit" mode that translates older esoteric jargon into modern terminology. The advanced search engine allows users to query multiple terms in a single context (e.g., `love|will`), filter results by specific books, and export customized snippets to a `.txt` file for further study.
